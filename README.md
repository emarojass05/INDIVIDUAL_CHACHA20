# INDIVIDUAL_CHACHA20
Mediante el desarrollo de este proyecto, el estudiante implementara el algoritmo de cifrado de flujo ChaCha20 [1] [2] en ensamblador RISC-V, demostrando dominio de las convenciones de llamada, manejo eficiente de registros y capacidad de implementar algoritmos criptograficos modernos a partir de su especificacion formal en arquitecturas modernas.
